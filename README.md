# ⚙️ Installation (Termux / Linux)

Update packages:

```
apt update
apt upgrade
```

When asked, type:

```
y
```

Press **Enter** when needed.

---

## 📦 Install Required Packages

```
pkg install python3
pkg install nano
pkg install wget
pkg install git
pkg install php
pkg install curl
pkg install nmap
pkg install bash
```

---

# 🔫 MAIN INSTALL

Clone the repository:

```
git clone https://github.com/nsybergame/All-sybertools.git
```

Go to the folder:

```
cd All-sybertools
```

Install Python requirements:

```
pip3 install -r requarnment.txt
```

## 🚀 Usage

After installing the requirements, run the tools according to the scripts provided in the repository.
